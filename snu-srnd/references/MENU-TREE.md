## 동적 열거

메뉴는 변경될 수 있으므로 항상 런타임에 확인한다:

```js
const ds = nexacro.Application.mainframe.VFrameSet.TopFrame.form.ds_menuAllList;
for (let r = 0; r < ds.getRowCount(); r++) {
  const id = ds.getColumn(r, 'MENU_ID');
  const nm = ds.getColumn(r, 'MENU_NM');
  const up = ds.getColumn(r, 'UP_MENU_ID');
  const pgm = ds.getColumn(r, 'PGM_ID');
  const start = ds.getColumn(r, 'START_MENU_ID');
}
```

또는 `scripts/srnd-helpers.js`의 `searchMenu(keyword)` 함수로 키워드 검색이 가능하다.
