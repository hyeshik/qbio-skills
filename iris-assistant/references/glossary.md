# IRIS / 국가 R&D 용어집

이 파일은 스킬 사용 중 학습한 도메인 용어를 누적 기록한다. 새 용어를 배울 때마다 여기에 추가하여 다음 작업에서 웹 검색 없이 바로 참조할 수 있게 한다.

---

## 3책5공 (3責5共)
- **정의**: 국가연구개발혁신법 시행령 제64조에 따른 동시수행 과제 수 제한 제도
- **기준**: 연구책임자로 최대 **3개**(3책), 전체(책임+참여) 합계 최대 **5개**(5공)
- **예외 대상**: 연평균 연구비 6천만 원 이하 소규모 과제, 교육훈련 과제, 자율운영 중점연구소, 기술사업화·창업 R&D 과제(2026년~)
- **IRIS 확인 방법**: 마이R&D 페이지 > "참여 과제 목록" 영역 바로 위 **"3책5공 확인"** 버튼 클릭 → NRI 기준 팝업 표시
- **참고**: 마이R&D 과제 목록(전체 참여 과제)과 3책5공 집계 과제 수가 다를 수 있음 — 예외 대상 과제는 집계에서 제외됨
- **학습일**: 2026-03-15
