# IRIS R&D 업무포털 전체 메뉴 맵

이 문서는 IRIS R&D 업무포털(https://www.iris.go.kr/resources/nui/index.do)의 전체 메뉴 구조를 정리한 참조 문서이다.

## 상단 메뉴 구조

포털 상단에 8개의 메인 메뉴 탭이 있다:

**워크라운지 | 사업 기획·공고 | 과제접수 | 과제수행 | 사후관리 | 과제평가 | 납부 | R&D 고객센터**

---

## 1. 워크라운지

사용자의 대시보드 영역. 과제 현황 요약, 알림, 증명서 발급 등을 제공한다.

| 메뉴 | 설명 | 주요 사용 시나리오 |
|------|------|-------------------|
| 마이 R&D | 과제 관리 현황 대시보드, 참여 과제 목록. **인라인 버튼**: "3책5공 확인" (참여 과제 목록 바로 위, NRI 기준 팝업) | "내 과제 현황 보기", "참여 중인 과제 확인", "3책5공 확인" |
| 전자알림 수신 목록 | IRIS에서 발송한 전자알림 확인 | "알림 확인", "전자알림 보기" |
| 국외수혜정보 보고 | 국외 수혜정보 신고 | "국외수혜정보 보고", "해외 수혜 신고" |
| 증명서 발급 | 과제 관련 증명서 발급 | "증명서 발급", "과제 증명서" |

## 2. 사업 기획·공고

| 메뉴 | 하위 메뉴 | 설명 |
|------|----------|------|
| 수요조사 | 정기 수요조사 | 정기 수요조사 참여 |
| | 수시 수요조사 | 수시 수요조사 참여 |
| 인터넷 공시/공청회 | | 인터넷 공시 및 공청회 정보 |
| 공모 예고 | | 향후 공모 예정 사업 확인 |
| 사업 공고 | | 현재 접수 중인 사업 공고 목록 |
| 사업설명회 | | 사업설명회 일정 및 정보 |

## 3. 과제접수

신규 과제 접수(신청) 영역.

| 메뉴 | 설명 |
|------|------|
| 신청 안내 | 과제 신청 절차 안내 |
| 신청공고목록 | 신청 가능한 공고 목록 및 접수 진행 |

## 4. 과제수행

진행 중인 과제의 협약, 연구비, 성과 등을 관리하는 핵심 영역. 연구책임자가 가장 많이 사용한다.

| 메뉴 | 하위 메뉴 | 설명 |
|------|----------|------|
| **협약신청** | 협약절차안내 | 협약 절차 설명 |
| | (협약용)연구개발계획서 제출 | 협약용 연구개발계획서 작성/제출 |
| | (협약용)연구개발계획서 신규 | 신규 연구개발계획서 작성 |
| | 개인정보연구윤리동의 | 개인정보 및 연구윤리 동의 절차 |
| | 협약신청 | 협약 신청 처리 |
| **협약변경** | (기관일괄)협약변경신청 | 기관 일괄 협약변경 |
| | (승인통보)협약변경신청 | 승인통보 방식 협약변경 |
| | 차년도협약변경신청 | 차년도 협약변경 |
| | 이전연차협약변경신청 | 이전 연차 협약변경 |
| **연구비** | 연구비 지급현황 | 연구비 지급 상태 조회 |
| | 부담금 납부현황 | 부담금 납부 상태 조회 |
| | 단계 이월금 신청 | 단계 간 이월금 신청 |
| **연구시설장비** | 연구시설장비 진행현황 | 연구시설장비 등록/관리 현황 |
| **성과** | 성과등록 | 연구 성과 등록 (논문, 특허 등) |
| | 안전기술보고서 제출 | 안전기술보고서 작성/제출 |
| **제재처분평가단 심의** | 제재심의 결과조회 | 제재심의 결과 확인 |
| | 재검토 요청 | 제재심의 결과 재검토 요청 |

## 5. 사후관리

| 메뉴 | 하위 메뉴 | 설명 |
|------|----------|------|
| **정산** | 정산결과 조회 | 정산 결과 확인 |
| | 이의신청 | 정산결과 이의신청 |

| | 청년고용 정보 및 실적제출 | 청년고용 관련 실적 제출 |
| | 종료 후 성과소유기관 변경 신청 | 성과소유기관 변경 |
| | (징수)기술료 사용실적 보고서 제출 | 징수 기술료 사용실적 보고 |
| | (비징수)기술료 사용실적 보고서 제출 | 비징수 기술료 사용실적 보고 |

## 의도 → 메뉴 경로 매핑

연구책임자(PI)가 가장 자주 하는 업무와 대응 메뉴:

| 사용자 의도 (자연어) | 목표 메뉴 경로 | 유형 |
|---------------------|---------------|------|
| "과제 접수/신청하려고" | 과제접수 > 신청공고목록 | 작업 |
| "연구개발계획서 작성/제출" | 과제수행 > 협약신청 > (협약용)연구개발계획서 제출 | 작업 |
| "협약 신청" | 과제수행 > 협약신청 > 협약신청 | 작업 |
| "협약변경 하고 싶어" | 과제수행 > 협약변경 > (승인통보)협약변경신청 | 작업 |
| "차년도 협약변경" | 과제수행 > 협약변경 > 차년도협약변경신청 | 작업 |
| "연구비 확인" | 과제수행 > 연구비 > 연구비 지급현황 | 조회 |
| "이월금 신청" | 과제수행 > 연구비 > 단계 이월금 신청 | 작업 |
| "성과 등록" | 과제수행 > 성과 > 성과등록 | 작업 |
| "보고서 제출" | 과제평가 > 보고서 > 보고서제출(연차/단계/최종) | 작업 |
| "정산결과 확인" | 사후관리 > 정산 > 정산결과 조회 | 조회 |
| "기술료 납부" | 납부 > 납부안내 > 기술료 납부안내 조회 | 조회/작업 |
| "기술료 사용실적" | 사후관리 > 기술료 > (징수)기술료 사용실적 보고서 제출 | 작업 |
| "증명서 발급" | 워크라운지 > 증명서 발급 | 작업 |
| "내 과제 현황" / "참여과제 목록" | 워크라운지 > 마이 R&D | 조회 |
| "알림 확인" | 워크라운지 > 전자알림 수신 목록 | 조회 |
| "평가결과 확인" | 과제평가 > 평가위원회 > 평가결과 단건조회 | 조회 |
| "연구시설장비" | 과제수행 > 연구시설장비 > 연구시설장비 진행현황 | 조회 |
| "사업 공고 확인" | 사업 기획·공고 > 사업 공고 | 조회 |
| "국외수혜정보" | 워크라운지 > 국외수혜정보 보고 | 작업 |
